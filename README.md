# BNK Finance — готовый сайт

Это полноценный сайт по вашему макету: стеклянный светлый дизайн, красный акцент BNK Finance, адаптивная вёрстка, анимации, кредитный калькулятор, форма заявки, переключение RU/KG и простая админ-панель для заявок.

## Структура

```text
bnk-finance-site/
├─ index.html                 # статическая версия для GitHub Pages / быстрого открытия
├─ admin.html                 # статическая страница панели
├─ public/
│  ├─ index.html              # версия для Node.js сервера
│  ├─ admin.html
│  ├─ css/styles.css          # дизайн
│  └─ js/app.js               # анимации, калькулятор, форма, язык
├─ data/applications.json
├─ package.json
├─ server.js
└─ README.md
```

## Почему раньше сайт был «голый»

HTML подключал файлы:

```text
public/css/styles.css
public/js/app.js
```

Но в архиве их не было. Поэтому браузер показывал только стандартный HTML без дизайна. В этой версии CSS и JS добавлены, а также добавлен `index.html` в корне для GitHub Pages.

## Быстрый просмотр без сервера

Откройте файл:

```text
index.html
```

Дизайн, анимации, калькулятор и форма будут работать. Без сервера заявки сохраняются в браузере через `localStorage`.

## Полный запуск с backend

```bash
npm install
npm start
```

Откройте сайт:

```text
http://localhost:3000
```

Админ-панель:

```text
http://localhost:3000/admin.html
```

Токен по умолчанию:

```text
change-me
```

## Публикация на GitHub Pages

Можно загрузить весь проект в репозиторий и включить Pages из ветки `main`, папка `/root`. GitHub Pages откроет корневой `index.html` и дизайн будет виден.

Важно: GitHub Pages не запускает Node.js. На GitHub Pages форма будет работать как демо и сохранять заявки только в браузере. Для реального сохранения заявок используйте Render, Railway, VPS, Beget Node.js или другой Node.js-хостинг.

## Публикация с полным функционалом

Для полного функционала нужен Node.js-хостинг. На хостинге укажите:

```bash
npm install
npm start
```

Переменные окружения:

```text
PORT=3000
ADMIN_TOKEN=ваш_секретный_токен
```
